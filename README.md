# 🐧 SysAdmin Scripts

Это мой первый набор скриптов для автоматизации повседневных задач системного администратора.  
Я учусь на третьем курсе и активно развиваюсь в администрировании Linux.

---

## 🛠️ Стек

- **Bash** — написание скриптов
- **Linux (Ubuntu/Debian)** — тестирование и выполнение
- **Git** — контроль версий

---

## 📂 Содержимое репозитория

| Файл | Описание |
|------|----------|
| `sys_info.sh` | Показывает текущее состояние системы: загрузку CPU, свободную память и место на диске. Полезно для быстрой диагностики. |
| `backup.sh` | Создаёт архив с указанием текущей даты в имени файла. |
| `firewall.sh` | Настраивает базовые правила файрвола: разрешает SSH, HTTP, HTTPS, блокирует всё остальное. |
